Exploring EEG signals to understand brain activity

Note on EEG Analysis

This is one of my early explorations of EEG data analysis using OpenNeuro datasets. As I am still learning how to decode and interpret EEG signals, some results may not be perfectly refined. This project represents my ongoing development in EEG data processing.

What is EEG Analysis?

EEG (Electroencephalography) is a technique used to measure electrical activity in the brain. By analyzing different brainwave frequencies, we can gain insights into states like relaxation, deep sleep, and active thinking.

EEG Data Processing Steps

  1. Loaded raw EEG data from OpenNeuro datasets.
  2. Filtered EEG signals to remove noise and artifacts.
  3. Computed Power Spectral Density (PSD) to observe brainwave activity.
  4. Visualized frequency bands to analyze cognitive states.

Code Used for EEG Analysis

This Python script was used to process and analyze the EEG data.


import mne
import matplotlib.pyplot as plt

# Load EEG data from OpenNeuro dataset
raw_data = mne.io.read_raw_brainvision('path/to/your/data.vhdr', preload=True)

# Apply band-pass filter (1-50 Hz) to remove noise
raw_data.filter(1, 50, fir_design='firwin')

# Compute Power Spectral Density (PSD) for Alpha band (8-12 Hz)
psd, freqs = mne.time_frequency.psd_welch(raw_data, fmin=8, fmax=12, n_fft=256)

# Plot the PSD
plt.plot(freqs, psd.mean(axis=0))
plt.title('PSD Alpha Band (8-12 Hz)')
plt.xlabel('Frequency (Hz)')
plt.ylabel('Power (dB)')
plt.show()
