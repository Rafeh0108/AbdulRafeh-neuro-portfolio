Stroke fMRI Analysis

My early exploration into neuroscience and coding using publicly available fMRI data.

Project Overview

This project was my first attempt at analyzing stroke fMRI data using OpenNeuro datasets. My goal was to visualize brain activity changes before and after rehabilitation therapy.

Changes in Brain Activity (Pre vs Post Therapy)

The heatmap below represents the voxel-wise difference between post- and pre-therapy fMRI scans. Red regions indicate increased activity post-therapy, while blue regions indicate decreased activity.

Statistical Analysis of Changes

This histogram quantifies how much the BOLD signal changed across all voxels. The average change was ~1.65, meaning there was a slight overall increase in activation, with some regions showing more extreme changes.

How I Did This Analysis

This project involved several key steps:

  • Finding a relevant fMRI dataset from OpenNeuro.
  • Using Python (Nilearn, Nibabel, Matplotlib) to process and visualize the data.
  • Extracting slices from 4D fMRI data and comparing voxel-wise differences.
  • Creating a histogram to summarize changes in brain activity.

What Didn’t Work

Not everything went smoothly! Some major setbacks I faced:

  • Struggled to correctly align pre- and post-therapy MRI scans, leading to some misaligned comparisons.
  • Failed attempts at visualizing the lesion location due to mismatched affine transformations.
  • Initially, my voxel-wise change visualization wasn’t highlighting differences correctly.
  • Spent too much time debugging color mapping instead of focusing on meaningful interpretations.

These challenges taught me a lot about neuroimaging, affine transformations, and the importance of checking my data step-by-step before visualizing results.

Future Directions

This was just my first step into neuroimaging analysis. Moving forward, I’d like to:

  • Improve lesion detection by using machine learning-based segmentation techniques.
  • Refine the pre/post MRI alignment to ensure voxel comparisons are accurate.
  • Explore how different types of rehabilitation therapies impact brain recovery over time.
