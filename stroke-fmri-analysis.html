    Stroke fMRI Analysis

    My early exploration into neuroscience and coding using publicly available fMRI data.

    Project Overview

    This project was my first attempt at analyzing stroke fMRI data using OpenNeuro datasets. My goal was to visualize brain activity changes before and after rehabilitation therapy.

    Challenges & Future Work

    While we made some progress, we faced issues with lesion localization and full therapy interpretation.

    • We couldn't precisely align the lesion mask with the MRI.
    • Finding detailed information on the exact rehabilitation therapy was challenging.
    • This was an early step in my journey into neuroscience data analysis.

    Next steps include improving alignment techniques and exploring stroke therapy details further.
